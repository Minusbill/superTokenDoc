# SuperToken 是什么

SuperToken 是一个面向开发者和团队的 AI 网关服务，用来统一接入主流大模型并给常见 AI 工具提供稳定的 API 入口。

## 你可以用它做什么

- 给 Claude Code、Codex、Gemini CLI 统一配置一个网关地址
- 给团队成员分发不同权限和配额的 API Key
- 按模型、项目和成员统计费用
- 在一个控制台中维护多个模型供应商

## 工作方式

```text
你的开发环境 -> SuperToken -> 模型供应商（OpenAI / Anthropic / Google 等）
```

这种方式的主要价值：

- 网络与可用性统一治理
- 兼容接口统一治理
- 计费与权限统一治理
- 团队接入成本更低

## 安全与隐私

- 不保存代码正文
- 只做鉴权、转发和配额控制
- 推荐使用子 Key 做成员级隔离
- 支持按环境分离测试和生产流量

## 适用对象

### 个人开发者

- 快速启用 AI 编程工具
- 不想分别接多个厂商
- 需要更清晰的消费统计

### 技术团队

- 统一 AI 接入规范
- 统一预算和审计
- 给成员分配不同额度

### 企业用户

- 需要稳定可控的统一网关
- 需要团队权限与结算能力
- 需要私有化或专属线路扩展
