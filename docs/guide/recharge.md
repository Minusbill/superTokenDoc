# 充值与退款

这一页是公开站里最需要替换成真实业务信息的部分。

## 充值流程

推荐文案结构如下：

1. 登录控制台
2. 进入余额中心
3. 选择充值金额
4. 完成支付并等待到账

## 你应该明确写清楚

- 最低充值金额
- 支持的支付方式
- 是否自动到账
- 人工充值的处理时段
- 开票门槛

## 推荐模板

```text
充值说明：
- 最低充值金额：请以官网说明为准
- 支付方式：请以官网说明为准
- 到账时间：请以官网说明为准
- 发票政策：请以官网说明为准
```

## 退款建议

如果支持退款，建议提前写清楚：

- 哪些情况可退
- 哪些情况不可退
- 退款周期
- 退款手续费是否扣除

## 风险提示

公开站不要写模糊表述，比如“原则上可退”“特殊情况另议”，这种内容后续容易引发争议。
