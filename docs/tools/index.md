# 工具概览

SuperToken 的目标不是要求用户换工具，而是让现有工具都能接到同一个网关。

## 接入前准备

1. 安装 Node.js
2. 创建 `SUPERTOKEN_API_KEY`
3. 确认你的网关地址

```bash
export SUPERTOKEN_BASE_URL="https://supertoken.cc/v1"
export SUPERTOKEN_API_KEY="sk-xxxxx"
```

## 已提供文档的工具

- Claude Code
- OpenClaw
- Codex
- Gemini CLI
- Droid CLI
- OpenCode
- CC-Switch

## 选择建议

- 偏代码生成与重构：看 [Claude Code](/tools/claude-code/)
- 偏命令行代理与多工具切换：看 [Codex](/tools/codex/) 与 [CC-Switch](/tools/cc-switch/)
- 偏自建或开源工具链：看 [OpenClaw](/tools/openclaw/) 与 [OpenCode](/tools/opencode/)
