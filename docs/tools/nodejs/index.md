# 安装 Node.js

多数 CLI 工具都依赖 Node.js 运行环境。

## 推荐版本

- Node.js 20 或更高
- npm 10 或更高

## 验证命令

```bash
node --version
npm --version
```

## 安装后检查

确认以下命令在终端可用：

```bash
node -v
npm -v
npx -v
```
